css이란 무엇인가?

(Cascading Style Sheets)

css 즉, (Cascading style sheets)는 웹 페이지의 디자인과 스타일을 정의하기 위한 스타일 시트 언어입니다.

프로그래밍 언어가 아니라 마크업 정보를 표현하는 마크업 언어로 문서의 내용 이외의 문서의 구조나 서식 같은 것을 포함한다. 보면 알겠지만 애초에 이름 HTML의 ML이 마크업 언어라는 뜻이다. 웹사이트에서 흔히 볼 수 있는 htm이나 html 확장자가 바로 이 언어로 작성된 문서다.

HTML과 함께 사용하여 웹 페이지의 레이아웃, 색상, 폰트, 간격 등을 조정할 수 있습니다. CSS는 HTML과는 별도의 파일로 작성됩니다. 이 파일은 .css 확장자를 가지며, HTML 파일 내에서 태그 내에 태그를 사용하여 호출됩니다. CSS는 선택자와 속성(property)의 집합으로 구성되며, 선택자는 스타일을 적용할 HTML 요소를 선택하고, 속성은 해당 요소에 적용될 스타일을 정의합니다. CSS를 사용하여 웹 페이지를 꾸미면 HTML과 CSS를 분리하여 유지보수와 변경이 용이해지며, 웹 페이지의 디자인을 일관성 있게 유지할 수 있습니다. 또한, CSS는 반응형 웹 디자인에도 중요한 역할을 합니다.
